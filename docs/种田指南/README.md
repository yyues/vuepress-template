# 种田流指南

假如穿越到了异世界，并且恰好是种田流呢？

那么如何生存呢？

> 重点： 提供的不是很全面

## 基础知识

发现大部分的种田流都可以分为以下种类

- 种田的地方很寒酸，科技树都没有点亮
- 人民愚昧无知，存在明显的阶级矛盾和地域歧视
- 不同于现在的地球环境，可能存在   **`魔幻类物品`**（PS：不然这就叫穿越了 哈哈）

## 重点方向

还是可以笼统的归为两类

- 人口-----不如说是人口红利
- 科技

当人口不足的时候，干什么都是比较费劲的，比如基建、战争都是让人痛苦的，在非现代化战争的背景下，人口都是关键

其中，**普及教育**和**思想改造**都是跑不掉的。更好的接受 工业化 和 科技化 才是 真正的人口红利。

科技树的点亮都是基本设备的迭代优化再发明而已。基建做好 基本就没毛病。
